=========================================
软件安装：UE4
=========================================

通过Epic安装UE4一般限于网速比较困难，可以网络上下载离线安装包，解压完会有四个文件夹。

.. code-block:: python

    Engine
    FeaturePacks
    Samples
    Templates

直接找下面的.exe文件打开运行即可。

.. code-block:: python

    Engine\Binaries\Win64\UE4Editor.exe

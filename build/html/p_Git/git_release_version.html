Git发布版本¶

Github中找到想要发布版本的repo代码仓库，找到Release选项创建新版本。

Gitlab中找到想要发布版本的repo代码仓库，点击Tag标签，New Tag，修改Tag name，Message以及Release notes，Tag name建议是v1.1.0这样的版本号。

  • 语义化版本标准：https://semver.org/lang/zh-CN/

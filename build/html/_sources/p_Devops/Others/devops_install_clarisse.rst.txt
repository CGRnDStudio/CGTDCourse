=========================================
软件安装：Clarisse
=========================================

- 选择isotropix_clarisse_ifx_4.0_win64.exe，右键以管理员身份运行。
- 一直Next傻瓜式安装，遇到python安装可以先不安装（如果安装过python27的话）。
- 修改配置文件C:\Program Files\Isotropix\Clarisse iFX 3.0\Clarisse\clarisse.env

.. code-block:: python

    PYTHONHOME=C:\Python27

Deadline渲染农场安装服务启动错误¶

这是一个典型的常见安装错误 Error starting service Deadline10DatabaseService , 遇见过几次，问题都是出在杀毒软件上，可以将杀毒软件暂时退出再进行安装，安装完再启动杀毒软件，我这边用到了火绒杀毒，关掉就好了。
